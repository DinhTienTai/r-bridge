Download R Bridge for ArcGIS

Current build: 1.0.1.240:

  • Download for R 4.0 (x64) arcgisbinding_1.0.1.240.zip
  • Download for R 3.6 (i386, x64) arcgisbinding_1.0.1.240.zip
Install latest version:
> install.packages("arcgisbinding", repos="http://r-arcgis.github.io/r-bridge", type="win.binary")
Check and update to latest version:
> update.packages(repos="http://r-arcgis.github.io/r-bridge", type="win.binary")
