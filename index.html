R Bridge for ArcGIS Pro 1.1+ and ArcGIS 10.3.1+

Current build:

Requirements:

R-3.6+ for Windows (https://cran.r-project.org/bin/windows/base/)

Installation command from the R Console:

> install.packages("arcgisbinding", repos="http://r-arcgis.github.io/r-bridge")

Update command:

> update.packages(repos="http://r-arcgis.github.io/r-bridge")
